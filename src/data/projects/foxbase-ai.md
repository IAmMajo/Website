Diese Website ist im Rahmen des Moduls „Webentwicklung“ im fünften Semester
meines Studiums in Zusammenarbeit mit anderen Studierenden entstanden. Es
handelt sich um ein Frontend für die Semantic Search API der FoxBase GmbH und
war ein exploratives Projekt zur Entwicklung eines UIs für AI. Das UI kann über
ein Admin-Panel umfangreich konfiguriert werden. Als Framework wurde
[Nuxt](https://nuxt.com/) verwendet.
